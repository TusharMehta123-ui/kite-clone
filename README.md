# Kite Clone

A pixel-perfect clone of [Zerodha Kite](https://kite.zerodha.com) — India's most popular trading platform UI.

Built with pure HTML + CSS. No frameworks, no dependencies.

## Features

- Dark-themed trading dashboard
- Watchlist with live-style price tickers
- Intraday candlestick chart area
- Buy/Sell order panel with margin info
- Fully responsive layout

## Run

Just open `index.html` in a browser — no build step needed.
